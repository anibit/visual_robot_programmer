Known Issues

  • Supported Browsers
    • Firefox 29+ (Tested on Windows)
    • Chrome 39+ (Tested on Windows, Android, and ChromeOS)
    • Safari XX (Tested on Mac OSX 10.8)
    • Internet Explorer 11+ (see note)
  • Internet Explorer is broken. - almost works on IE, but there are some major usability issues.
  • Functions the return values. - the resulting code will have errors
  • Local variables - All variables defined in your code (except for function parameters) are currently global variables. There are plans to support locally declared variables eventually.
  • No support for floating point types. We would like to support them eventually.
  • There are currently limited(no) options for converting numbers to text.
  • The user interface can be difficult to use on mobile devices, we want to fix that eventually.

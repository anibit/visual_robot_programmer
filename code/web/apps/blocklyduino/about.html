The Anibit Graphical Programming Tool is provided free of charge by Anibit Technology, LLC.

This is released in hopes that people will find it useful for helping to teach programming and robotics to beginners.

Do you run Windows? You can get a Windows version of this tool that makes working with the 3Pi easy easy as pushing a button, great for education and demos. Get it for free!

For a list of known bug and issues, please see known issues.

You may download the source to this application from GitHub.
